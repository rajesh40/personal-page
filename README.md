# Portfolio Deploy Folder

This folder contains only public portfolio content for hosting.

Private interview docs are intentionally excluded.

## GitHub Pages setup

1. Push this repository to GitHub.
2. Open Settings -> Pages.
3. Set Build and deployment to Deploy from a branch.
4. Select your main branch and choose the /site folder.
5. Save. GitHub Pages will publish only this folder.

https://rajesh40.github.io/personal-page

## Local preview

1. Open terminal in `C:\projects\tutorials\tuits`.
2. Run `py -m http.server 8080` (or `python3 -m http.server 8080` in WSL/Linux).
3. Open `http://localhost:8080/site/bagging.html` in your browser.
