# Media Guide

Use this folder for screenshots, diagrams, and short demo assets.

Recommended structure:

- `assets/media/vehicle-inspection/`
- `assets/media/rule-manager/`
- `assets/media/batch/`
- `assets/media/order-integrations/`
- `assets/media/bagging/`
- `assets/media/chargebee-cart/`

Recommended file types:

- `.png` or `.webp` for screenshots
- `.mp4` only if small and necessary

Better approach for videos:

- upload videos to Loom, YouTube, or Vimeo
- paste the public links into the project pages

Do not upload:

- client data
- private code screenshots
- secrets
- internal-only dashboards with sensitive data visible
