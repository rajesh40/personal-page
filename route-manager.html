Independent Project

Route Manager

Route Manager was a map-driven operational workspace for planning routes, reviewing shipments, running optimization, and handling controller-side route actions.

Visual Overview

Reference visual of the route planning workspace and controller-facing route operations.

Why It Matters

This is one of the strongest full-stack stories because it combines map UI, route lifecycle, real-time updates, and state-heavy workflow coordination.

Tech Stack

  • Next.js operational UI
  • Redux Toolkit state orchestration
  • map-based workflow and route planning
  • MQTT-driven live updates

What I Worked On

  • route optimization and controller review flow
  • shipment selection, side panels, drawers, and route state sync
  • save, assign, merge, and update route workflows
  • operational actions such as reschedule, cancel, and address updates
