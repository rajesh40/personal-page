Media Guide

How to add screenshots and videos without exposing company-sensitive information.

Use this page as the operating guide for filling the portfolio. The goal is to make projects easier to understand while keeping the portfolio clean, safe, and recruiter-friendly.

Recommended Media Per Project

  • 1 overview screenshot
  • 1 workflow detail screenshot
  • 1 configuration or admin screenshot
  • 1 short demo video link, 60 to 120 seconds

Best Video Format

  • Use Loom for short explainers.
  • Use YouTube only if you want public discoverability.
  • Keep the video focused on workflow and architecture, not code ownership.

Do Not Upload

  • private code
  • customer data
  • credentials
  • internal-only dashboards with live sensitive values

What To Show Instead

  • blurred or sanitized screenshots
  • demo data
  • architecture sketches
  • workflow-focused screen recordings
